Dataset Used:
This repo uses the Trip Advisor Hotel Reviews dataset, a public Zenodo dataset that can be found at: https://zenodo.org/record/1219899#.Y2QVVOzMJeg
This dataset contains 20491 raw text reviews from Tripadvisor and their ratings (from 1-5 stars).

Citation - Barkha Bansal. (2018). TripAdvisor Hotel Review Dataset [Data set]. Zenodo. https://doi.org/10.5281/zenodo.1219899. Licensed under the Creative Commons Attribution-NonCommercial 4.0 International Public License.

This dataset originated from the following paper: https://zenodo.org/record/1219899#.Y2QZ0uzMIeY
